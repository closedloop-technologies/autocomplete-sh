Autocomplete.sh

Hack your terminal workflow with AI-powered suggestions

Quick Start 

wget -qO- https://autocomplete.sh/install.sh | bash
Initialize Autocomplete.sh

What is Autocomplete.sh?

Autocomplete.sh injects AI-powered command-line suggestions directly into your terminal. Just type TabTab to summon the power of LLMs and get context-aware recommendations.

Intuitive

Use natural language without context switching

Context-Aware

Considers your terminal state and command history

Flexible

Supports various LLM models to fit your needs

Efficient

Faster than searching Stack Overflow or ChatGPT

How It Works

Autocomplete.sh leverages your terminal's context to provide accurate suggestions:

  • Machine info: $USER, $PWD, $HOME, $OSTYPE, $BASH, $TERM, $HOSTNAME
  • Environment variables (names only)
  • Command history
  • Current directory contents
  • Command-specific help information

To see the full prompt:

autocomplete command --dry-run "your command here"

Configuration

View current settings:

autocomplete config

Change a setting:

autocomplete config set <key> <value>

Example: autocomplete config set api_key sk-your-api-key

Reset to defaults:

autocomplete config reset

Usage Tracking 

autocomplete usage

Track your API calls and associated costs. The average cost is about $0.005 per request using the GPT-4 model.

Cost-Saving Tip

Switch to the GPT-3.5-turbo model for lower costs (about $0.0005 - $0.001 per request): 

autocomplete config set model gpt-3.5-turbo
autocomplete config set api_prompt_cost 0.0000005
autocomplete config set api_completion_cost 0.0000015
